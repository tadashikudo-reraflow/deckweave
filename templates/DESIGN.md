## Visual Rules

- Use visuals that connect directly to the core message.
- Prefer information hierarchy over decoration.
- Keep editable text as real text boxes where possible.

## Do

- Use consistent spacing and type scale.
- Keep the main message visible at scan speed.
- Preserve brand color roles.

## Don't

- Use unrelated astronaut, abstract orb, or generic AI imagery.
- Hide weak content behind gradients.
- Rasterize text that should remain editable.
